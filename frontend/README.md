# DealIQ — Frontend

React 18 + TypeScript + Vite + Tailwind CSS + Shadcn UI

See the [root README](../README.md) for full project documentation, setup instructions, and API reference.

## Local Dev

```bash
npm install
npm run dev
```

Frontend runs at `http://localhost:5173`. Requires backend running at `http://localhost:8000` (or set `VITE_API_URL`).

## Environment Variables

```env
VITE_API_URL=http://localhost:8000
```
